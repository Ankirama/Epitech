Choice Of Components¶

During this project we chose somes components to create our project:

  • JDK version used
  • Libraries used

JDK version used¶

We used the jdk 1.7.X version, from windows (at the beginning) and Mac OSX Yosemite.

This version was unable to handle lambda version so we used reflexivity.

Libraries used¶

We used some libraries for this project:

  • AWT Library
  • Swing Library
  • Math Library
  • Util Library

AWT Library¶

This library allow us to create components in our panel (GUI). In this Library we used GridBagLayout and GridBagConstraints

Swing Library¶

This library allow us to create panels and frames (JPanel / JFrame) We used both and create child classes to allow to use methods from these classes. It allow ut more flexibility too.

Math Library¶

This library was useful to get some Math functions like *sqrt* or *ln*

Util Library¶

This library was our Collection tools. We used some Collections like:

  • Map
  • HashMap
  • Deque
  • ArrayList
  • ...
